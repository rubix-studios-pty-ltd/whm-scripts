# WHM Scripts

Collection of WHM Scripts to enable quick customisation and extension to existing WHM installations.

## Script

### Contact Providers

Collection of WHM iContact provider, allowing WHM to send server contact notifications through the selected provider.

**[Ntfy](https://ntfy.sh)**

```
curl -fsSL https://raw.githubusercontent.com/rubix-studios-pty-ltd/rubix-notify/main/install.sh | bash
```
